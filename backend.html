Sistema Orientado a Objeto

Sistema backend desenvolvido em Java como projeto acadêmico da disciplina de Programação Orientada a Objetos, com foco na modelagem completa de um domínio real: uma academia com múltiplas salas, clientes, colaboradores, agendamentos, vendas e controle financeiro.

O projeto abrange desde a análise de requisitos até a implementação orientada a objetos, aplicando conceitos fundamentais como encapsulamento, herança, polimorfismo, interfaces, uso de coleções, persistência em JSON e boas práticas de organização de código.

Foram implementados fluxos completos de cadastro e gerenciamento de clientes, funcionários e produtos, controle de agendamentos (diárias e mensalidades), geração de relatórios, balanço financeiro mensal e integração lógica com um sistema externo de catraca.

O sistema também inclui ordenação, busca e iteração de dados utilizando Iterator, Comparator, Collections.sort() e binarySearch(), com testes e explicações documentadas no código.
